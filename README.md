另外，PikPak 也有設置內容檢舉與審核機制，會針對被檢舉確認屬於極端有害內容的連結進行屏蔽,整體審核原則偏向「盡量少干預、尊重內容多樣性」，跟部分內容審查嚴格的雲端服務比起來相對寬鬆。

**PikPak 真的能保密我下載的內容嗎？**

資料傳輸採用 SSL/TLS 加密，網路服務商無法監看你下載的具體內容，這是它被稱為「私密雲盤」的主要依據。但要注意它不是端對端加密，服務商技術團隊理論上仍能存取伺服器內的檔案，極度敏感的個資不建議存放。
